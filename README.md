# xx-for-bbn 

声明：本仓库仅为自用备份，不适合别人使用，非开源项目，请勿擅自使用与传播，否则责任自负。

脚本特点：支持临时隧道，固定隧道，可在list.log中查看节点信息

使用方法：

1、第一步：把玩具原来的JAVA启动文件改名为senver.jar

2、第二步：上传仓库中的server.jar、c.yml文件到玩具中

3、第三步：修改c.yml中的参数信息

4、第四步：启动玩具即可


# 固定隧道设置：

vmess 端口  localhost:8001

vless 端口  localhost:8002
